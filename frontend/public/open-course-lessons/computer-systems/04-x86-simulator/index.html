看得见的\"取指→译码→执行→访存→写回\"

本演示是一个用纯 JavaScript 实现的 Y86-64 教学 CPU 模拟器，同时支持 SEQ（顺序）与 PIPE（5 级流水）两种执行模式，随程序运行同步高亮数据通路上的活跃模块与信号。学生可选择\"求和 / 数组累加 / 简单分支\"三个内置程序，逐周期或连续运行，观察 CPI、气泡、分支惩罚等真实指标。

🎯 教学目标：把 CSAPP Y86-64 Arch Lab 的 Tcl/Tk GUI 搬进浏览器；一次课上看到 SEQ 与 PIPE 在同一程序上的性能差异；直观理解流水线冒险与气泡插入。
