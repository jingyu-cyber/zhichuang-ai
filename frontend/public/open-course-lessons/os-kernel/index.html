操作系统内核构建

12 节交互式课程 — 从启动到文件系统，用动画理解内核原理

1 内核启动与系统调用

第 1 节 启动流程 bootloader → kernel entry → mm_init → clock → console，各阶段依次点亮时间线 启动时序 第 2 节 Trap 陷入机制 用户态 → trap → 内核态 → handler → 返回，寄存器/栈/特权级动画 中断特权级 第 3 节 添加系统调用 syscall 表条目 → dispatcher → 用户触发 → 内核处理 → 返回值 系统调用内核

2 进程管理

第 4 节 进程状态机 就绪/运行/阻塞/退出，点击事件触发状态转换 状态阻塞 第 5 节 CPU 调度器 就绪队列 → CPU → 时间片耗尽 → 回队，FCFS/SJF/RR 甘特图 调度时间片 第 6 节 上下文切换 进程 A 寄存器保存到 PCB → 进程 B PCB 恢复到寄存器 PCB切换

3 内存管理

第 7 节 虚拟地址空间布局 text/data/heap/stack/mmap 各段地址范围与权限 虚拟内存段 第 8 节 页表遍历 虚拟地址拆分 → 4 级页表逐级查找 → 定位物理页 MMU四级 第 9 节 缺页异常处理 访问未映射 → trap → 分配帧 → 更新页表 → 重试 异常缓存

4 文件系统与 I/O

第 10 节 文件描述符 fd 表 → 打开文件表 → inode 指针链，dup() 共享引用 fdinode 第 11 节 路径解析 /home/user/file.txt 逐级 directory entry → inode 查找 目录逐级 第 12 节 设备 I/O 请求 → 设备队列 → DMA 传输 → 中断 → 进程唤醒 DMA中断
