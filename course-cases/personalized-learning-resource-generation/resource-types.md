# 个性化资源类型

智创Agent 个性化学习方向首版至少生成 6 类资源。

| 资源类型 | 内容 | 个性化依据 |
|---|---|---|
| 讲解文档 | 针对知识点的分步说明 | 学生基础、目标方向 |
| 知识点思维导图 | 概念层级和学习顺序 | 当前短板、课程结构 |
| 练习题 | 选择题、简答题、代码题 | 易错点、目标难度 |
| 拓展阅读 | 推荐资料和阅读顺序 | 资源偏好、学习阶段 |
| 代码实操案例 | 小型代码任务或伪代码 | 项目目标、技术栈 |
| 项目实践材料 | 项目任务书、Rubric、验收标准 | 学习目标、项目经历 |

## 资源卡片字段

| 字段 | 说明 |
|---|---|
| 标题 | 资源名称 |
| 类型 | 文档、题目、实操、项目等 |
| 适合原因 | 为什么推荐给该学生 |
| 关联画像 | 对应画像维度 |
| 引用来源 | 知识库来源或项目证据 |
| 完成方式 | 阅读、练习、编码、提交项目 |
| 检查点 | 完成后应达到的结果 |
