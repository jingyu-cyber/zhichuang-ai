# 数据模型草案

本文件用于统一首版实体边界，后续实现时同步到 SQLAlchemy models 和 Alembic migration。

## 1. 用户与权限

### `users`

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| `id` | string | 用户 ID |
| `name` | string | 姓名 |
| `role` | enum | `student` / `teacher` / `admin` |
| `email` | string | 邮箱 |
| `student_no` | string | 学号，可为空 |
| `teacher_no` | string | 教工号，可为空 |
| `created_at` | datetime | 创建时间 |

### `course_memberships`

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| `id` | string | 记录 ID |
| `course_id` | string | 课程 ID |
| `class_id` | string | 班级 ID |
| `user_id` | string | 用户 ID |
| `role_in_course` | string | student / teacher / assistant / class |

## 2. 教学基础数据

### `courses`

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| `id` | string | 课程 ID |
| `name` | string | 课程名称 |
| `term` | string | 学期 |
| `description` | text | 课程说明 |

### `classes`

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| `id` | string | 班级 ID |
| `name` | string | 班级名称 |
| `grade` | string | 年级 |
| `major` | string | 专业 |

教学基础数据首版使用 SQLite 持久化，公开 Demo 首次启动会写入示例课程、班级、学生和课程成员关系。

## 7. 学习任务与复盘

### `learning_tasks`

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| `id` | string | 任务 ID |
| `student_id` | string | 学生 ID |
| `title` | string | 任务标题 |
| `source` | string | 来源：作业报告、成长路径、竞赛推荐或用户手动保存 |
| `status` | enum | todo / doing / done |
| `priority` | enum | low / medium / high |
| `due_date` | string | 截止日期 |
| `evidence_required` | text | 需要提交或保留的证据 |
| `progress` | integer | 进度百分比 |
| `created_at` | datetime | 创建时间 |
| `updated_at` | datetime | 更新时间 |

任务中心首版使用 SQLite 持久化，支撑“用户确认后保存任务”和“定期复盘读取未完成任务”的连续使用。
