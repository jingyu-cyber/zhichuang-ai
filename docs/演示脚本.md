# 智创Agent 演示脚本

## 1. 开场

智创Agent·计算机学科垂类大模型与双创能力赋能平台，面向学校真实教学场景，把课程项目、知识库、学生画像和竞赛创新能力培养连接成一个闭环。

## 2. 教师看板

使用教师演示账号“周老师”。

讲解点：

- 教师直接查看班级项目提交情况。
- 系统给出维度分布、共性问题和讲评建议。
- 教师端不是审核流程，而是消费系统生成的学情分析结果。

## 3. 项目报告

切换到“项目报告”。

讲解点：

- 报告包含多维度评分、能力证据、主要问题和下一步任务。
- 分数是基于证据的相对画像，不是绝对能力判断。
- 项目提交物沉淀为学生成长证据。

## 4. 成长规划

切换学生演示账号“林一舟”。

讲解点：

- 系统根据能力画像生成阶段学习计划。
- 同时给出竞赛推荐和组队推荐。
- 组队需求以结构化卡片发布，联系方式默认不公开。
- 推荐理由需要解释能力互补和准备风险。

## 5. 知识库问答

提问：“如何准备算法竞赛？”

讲解点：

- 回答来自首批知识库资料。
- 结果带引用来源。
- 后续可接入课程大纲、项目要求、Rubric、竞赛资料和项目案例。

## 6. 知识库管理

切换管理员演示账号“平台管理员”。

讲解点：

- 展示首批重点路径：算法竞赛、AI 应用开发、软件项目实践。
- 查看资料清单、标签和检索命中。
- 管理员维护资料，学生和教师使用资料。

## 7. 收束

一句话总结：

智创Agent 把“项目分析、教师学情、学生成长、知识库问答、竞赛双创推荐”做成同一个教学智能体平台，既能比赛演示，也能继续向学校真实使用演进。
