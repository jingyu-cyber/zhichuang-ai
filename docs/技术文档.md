# 智创Agent · 技术文档

## 系统架构

```
┌──────────────────────────────────────────┐
│          前端 React 18 + TypeScript       │
│       Ant Design + ECharts + Zustand      │
│            Vite 构建 · Nginx 部署          │
├──────────────────────────────────────────┤
│        Nginx 反向代理 (Gzip + 缓存)        │
├──────────────────────────────────────────┤
│       FastAPI 后端 (Python 3.11)          │
│   ┌─────────────────────────────────┐    │
│   │  LangGraph 多 Agent 工作流       │    │
│   │  RAG 检索增强生成管线             │    │
│   │  12 个 API 路由模块              │    │
│   └─────────────────────────────────┘    │
├──────────────────────────────────────────┤
│   SQLAlchemy ORM    ChromaDB 向量库       │
│   PostgreSQL / SQLite    Redis 缓存       │
├──────────────────────────────────────────┤
│   DeepSeek LLM    OpenAI-compatible       │
│   通义千问 ASR     DashScope API          │
└──────────────────────────────────────────┘
```

## 技术栈

| 层级 | 技术选型 | 说明 |
|------|---------|------|
| 前端框架 | React 18 + TypeScript | SPA 单页应用 |
| UI 组件 | Ant Design 5 | 企业级 UI 组件库 |
| 可视化 | ECharts 5 | 能力热力图、班级分布图 |
| 状态管理 | Zustand 4 | 轻量级状态管理 |
| 构建工具 | Vite 5 | 快速开发与构建 |
| 后端框架 | FastAPI | 异步高性能 API |
| ORM | SQLAlchemy 2 | 数据库抽象层 |
| AI 编排 | LangGraph | 多 Agent 工作流 |
| 向量数据库 | ChromaDB | RAG 知识检索 |
| 缓存 | Redis 7 | 会话与临时数据 |
| 数据库 | PostgreSQL 16 | 生产数据存储 |
| LLM | DeepSeek Chat | 通用大语言模型 |
| ASR | 通义千问 Paraformer | 语音识别 |

## 部署架构

```
                    ┌─────────────┐
    用户 ──────────→│  Nginx :80   │
                    └──────┬──────┘
                           │
              ┌────────────┼────────────┐
              ▼            ▼            ▼
         /assets/     /api/        /open-course-lessons/
         (静态文件)   (后端API)     (交互式课件)
              │            │            │
              ▼            ▼            ▼
         1年缓存      FastAPI:8000   静态HTML文件
```

### Docker Compose 服务

| 服务 | 端口 | 说明 |
|------|:--:|------|
| frontend | 5173→80 | Nginx + 静态资源 |
| backend | 8000 | FastAPI 后端 |
| postgres | 5432 | 数据库 |
| redis | 6379 | 缓存 |

## 部署优化

| 优化项 | 实现 | 效果 |
|------|------|------|
| Gzip 压缩 | Nginx `gzip on` | HTML/CSS/JS >60% 缩减 |
| 静态缓存 | `immutable` + 1年 | 二次访问零请求 |
| 安全头 | XSS/Frame/Content-Type | OWASP 基线 |
| 健康检查 | Docker HEALTHCHECK | 异常自动重启 |
| 多 Worker | uvicorn `--workers 2` | 多核利用 |

## API 设计

### 核心接口

| 模块 | 端点 | 方法 | 说明 |
|------|------|:--:|------|
| 认证 | `/api/auth/demo-accounts` | GET | 演示账号列表 |
| 认证 | `/api/auth/demo-session` | POST | 创建演示会话 |
| 项目管理 | `/api/assignments` | GET/POST | 项目发布与列表 |
| 项目分析 | `/api/assignments/{id}/reports/{sid}` | GET | 项目分析报告 |
| 成长规划 | `/api/students/{id}/profile` | GET | 学生能力画像 |
| 成长规划 | `/api/students/{id}/plans` | GET | 学习计划 |
| 竞赛 | `/api/competitions` | GET | 竞赛列表与推荐 |
| 知识库 | `/api/knowledge/documents` | GET/POST | 资料入库与检索 |
| 知识库 | `/api/knowledge/search` | GET | 语义搜索 |
| 教师看板 | `/api/evaluations/dashboard` | GET | 班级学情总览 |
| 教师看板 | `/api/teacher/candidate-screening` | POST | 梯队筛选 |
| 课程 | `/api/courses` | GET | 课程列表 |

### 认证与权限

系统支持三级角色权限控制：

| 角色 | 权限范围 |
|------|------|
| 学生 | 本人项目、画像、计划、任务 |
| 教师 | 授权班级的学生数据和项目报告 |
| 管理员 | 全部数据、知识库管理 |

## 数据模型

### 核心实体

```
User (用户)
  ├── Profile (能力画像)
  ├── Plan (学习计划)
  ├── Task (执行任务)
  └── TeamStatus (组队状态)

Course (课程)
  └── Class (班级)
       └── Assignment (项目)
            └── Report (分析报告)
                 ├── Score (评分)
                 ├── Finding (问题)
                 └── Evidence (证据片段)

Knowledge (知识库)
  ├── Document (资料)
  └── Search (检索记录)
```

## 开源课堂课件

8门课程共96节交互式课件，每节为独立HTML文件，核心技术特点：

| 特性 | 实现 |
|------|------|
| 键盘导航 | ← → 步骤切换，Space 自动播放，R 重置 |
| SVG 可视化 | TC/TL 自动字号适配引擎 |
| 零网络依赖 | 单文件 HTML，无外部资源 |
| 课堂投屏 | 独立全屏页面，preserveAspectRatio 自适应 |
