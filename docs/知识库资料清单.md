# 首批知识库资料清单

本清单是首版系统需求，不绑定具体整理人。资料可由不同成员整理，最终统一进入知识库管理流程。

## 1. 首批重点路径

1. 算法竞赛。
2. AI 应用开发。
3. 软件项目实践。

## 2. 数量要求

| 类型 | 首版最低数量 | 用途 |
| --- | ---: | --- |
| 课程资料 | 5 门 | 支撑课程问答、项目分析、学习路径 |
| 竞赛资料 | 10 条 | 支撑竞赛推荐、训练计划、组队建议 |
| 项目案例 | 10 个 | 支撑软件实践、AI 应用开发、报告示例 |
| Rubric | 3 套以上 | 支撑代码项目评分和解释 |
| FAQ | 20 条以上 | 支撑常见问题回答 |

## 3. 课程资料建议

- 程序设计基础。
- 数据结构。
- 算法设计与分析。
- 软件工程。
- 数据库系统。
- Web 应用开发。
- 人工智能导论。

每门课程建议包含：

- 课程大纲。
- 章节知识点。
- 典型项目要求。
- 评分 Rubric。
- 推荐资料。

## 4. 竞赛资料建议

- 蓝桥杯。
- ACM/ICPC 训练资料。
- 中国大学生计算机设计大赛。
- 中国国际大学生创新大赛。
- 挑战杯。
- 大学生服务外包创新创业大赛。
- 软件杯。
- 华为 ICT 大赛。
- 全国大学生信息安全竞赛。
- 互联网+ 校内训练资料。

每条竞赛资料建议包含：

- 竞赛简介。
- 适合学生类型。
- 能力要求。
- 准备周期。
- 推荐训练内容。
- 官方或校内资料来源。

## 5. 项目案例建议

- 课程项目管理系统。
- 校园二手交易平台。
- 智能问答知识库。
- RAG 文档问答系统。
- 代码评测与报告系统。
- 简历分析助手。
- 数据可视化看板。
- 小程序预约系统。
- AI 学习助手。
- 项目协作管理工具。

每个项目案例建议包含：
